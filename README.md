## Установка
:information_source: Для установки необходимо расширение **Stylus**: [Chrome][1], [Opera][2], [Firefox][3]
А так же наличие стиля **[Vanilia Dark 2][4]**

:package: **[Установить стиль][5]** в формате User CSS.
